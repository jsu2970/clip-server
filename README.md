# 의존성 설치 방법
1. 프로젝트 루트에서 .venv\Scripts\activate로 가상환경 진입
2. 가상환경 상태에서 pip install -r requirements.txt로 의존성 설치
3. VSCODE에서 상단의 검색바에 Python: Select Interpreter 입력 후, (.venv)가 붙은 python 3.X 선택

# 로컬에서 서버 실행 방법
1. 가상환경 진입 상태에서 src 폴더로 진입 후, python main.py -> 서버 실행 
2. main.py 파일을 열어둔 후, 상태에서 프로젝트 상단의 Run 버튼에서 start로 실행 -> 서버 실행
